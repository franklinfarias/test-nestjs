## Descrição

Como trabalhar com Nestjs

## Instalação

* Crie o volume do MySQL
``` bash
docker volume create iniciando-nestjs-dbdata
```
* Execute o projeto com o Docker:
``` bash
docker-compose up
```

## Rodar o projeto

* Acesse http://localhost:3000 para acessar o Nest.js
